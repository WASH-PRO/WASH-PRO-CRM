# Telegram-боты

См. полную документацию: [docs/telegram.md](../docs/telegram.md) (GitHub Pages).

## Кратко

- Управление: **Dashboard → Telegram** (PyOrchestrator + `pyorch-bridge`)
- Доступ сотрудников: **Пользователи → Telegram user_id** + группа RBAC
- Viewer — только просмотр; Operator — создание/команды; Administrator — полный доступ
- Посторонним — только сообщение «Частный бот»
- Шаблон бота v2.7: единые отчёты, режимы из справочника, lock по токену

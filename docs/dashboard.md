React-приложение с тёмной/светлой темой и адаптивным меню (гамбургер на мобильных).

## Разделы

| Раздел | Путь | Доступ | Описание |
|--------|------|--------|----------|
| Обзор | `/` | Все | Сводка по автомойкам и постам |
| Автомойки | `/washes` | Все | Справочник автомоек (без серийного номера) |
| Посты | `/posts` | Все | Посты с серийным номером контроллера |
| Состояние | `/states` | Все | SCADA — режим, паузы, связь |
| Карты | `/cards` | Все | Карты клиентов |
| Статистика | `/usage` | Все | Использование по периодам |
| Финансы | `/finance` | Все | Выручка, наличные/безнал |
| Архив | `/archive` | Все | Политики архивирования |
| Резервные копии | `/backups` | Admin | Список и ручной запуск бэкапа |
| Telegram | `/telegram` | Admin | Настройки бота |
| Уведомления | `/notifications` | Все | Системные уведомления |
| Логи | `/logs` | Admin | Аудит Dynamic API |

## Роли пользователей

| Роль | Права в Dashboard |
|------|-------------------|
| **Administrator** | Полный доступ, включая бэкапы, Telegram, логи |
| **Operator** | Просмотр, создание, редактирование |
| **Viewer** | Только просмотр |

Роли настраиваются в Dynamic API Panel через группы пользователей.

## Автомойки и посты

- **Автомойка** — название, описание, адрес, дата регистрации, флаг облака.
- **Пост** — привязка к автомойке, номер поста, название, **серийный номер** (идентификатор контроллера), статус.

Серийный номер используется `message-processor` для сопоставления телеметрии с постом.

## Сборка и разработка

```bash
cd dashboard
npm install
npm run dev      # http://localhost:5173
npm run build    # production в dist/
```

В Docker Dashboard проксирует `/api` на `dynamic-api:3001`.

## Технологии

- React 18 + TypeScript
- React Router
- Tailwind CSS
- Lucide Icons
- Vite

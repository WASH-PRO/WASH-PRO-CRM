## English

### Fixed

- **Bot link / QR** — `pyorch-bridge` resolves `@username` via Telegram `getMe` with IPv4-first DNS and an IPv4 HTTPS fallback (Docker often breaks AAAA routes; token was saved but username never registered).

### Upgrade

```bash
# Dashboard → Settings → Software updates → CRM
git fetch origin && git checkout v1.1.61
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build pyorch-bridge
```

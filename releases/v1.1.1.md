## 🇬🇧 English

### Fixed

- **Updater stuck on "Build and restart CRM"** — build step rebuilt `update-bridge` itself, killing the update process (**exit 137**) and freezing UI forever; `update-bridge` is now excluded from that step (updates on full `docker compose up -d --build`)
- **Interrupted update jobs** — stale `running` status blocked UI and new runs; on `update-bridge` start, interrupted jobs are marked failed and `activeJobId` cleared (`recoverInterruptedJobs`)

### Upgrade

```bash
git pull
docker compose up -d --build --no-deps update-bridge dashboard
```

Stuck jobs reset automatically when `update-bridge` restarts.

## 🇷🇺 Русский

### Исправлено

- **Зависание на «Сборка и перезапуск CRM»** — шаг `build` пересобирал `update-bridge`, процесс обновления убивался (**exit 137**), UI зависал; `update-bridge` исключён из этого шага
- **Незавершённые задачи** — статус `running` блокировал UI; при старте `update-bridge` прерванные задачи помечаются failed (`recoverInterruptedJobs`)

### Обновление

```bash
git pull
docker compose up -d --build --no-deps update-bridge dashboard
```

Зависшая задача сбросится при перезапуске `update-bridge`.

v1.1.31 — Modules page UI polish · Доработка UI страницы «Модули»

**English** · **Русский**

---

## 🇬🇧 English

### Changed

- **Modules page** — compact filter toolbar (same layout as DataTable); search field no longer stretches full width
- **Module cards** — icon-only action buttons with tooltips; clickable title and icon
- **Pagination** — default 4 per page; fixed slice logic for load-more
- **Refresh catalog** — text button in page header (icon + label)

### Upgrade

```bash
git pull
docker compose build --pull=false dashboard
docker compose up -d --no-deps dashboard
```

On Mac without Docker Hub: use `--pull=false` if base images are cached.

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/modules/

## 🇷🇺 Русский

### Изменено

- **Страница «Модули»** — компактная панель фильтров (как в DataTable); поле поиска не растягивается на всю ширину
- **Карточки модулей** — кнопки действий только иконками с подсказками; кликабельные заголовок и иконка
- **Пагинация** — по умолчанию 4 на странице; исправлена логика «загрузить ещё»
- **Обновить каталог** — текстовая кнопка в шапке (иконка + подпись)

### Обновление

```bash
git pull
docker compose build --pull=false dashboard
docker compose up -d --no-deps dashboard
```

На Mac без Docker Hub: `--pull=false`, если базовые образы в кэше.

Документация: https://wash-pro.github.io/WASH-PRO-CRM/ru/modules/

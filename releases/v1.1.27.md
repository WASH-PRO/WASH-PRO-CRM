## 🇬🇧 English

### Fixed

- **Help** — visible teal **Help** button in the header and **Help** item at the bottom of the sidebar
- **Stale dashboard cache** — nginx returns 404 for missing `/assets/*.js` instead of SPA HTML; clearer message after CRM rebuild

### Changed

- `index.html` — `no-cache` headers; `/assets/` — immutable cache

### Upgrade

```bash
git pull
docker compose up -d --build dashboard
```

If build fails on Docker Hub, restart only: `docker compose up -d --no-deps dashboard`

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/

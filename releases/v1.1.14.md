**English** · **Русский**

## 🇬🇧 English

### Fixed

- **Notifications list** — messages are shown from **prepared i18n templates by event type**, not the raw text stored in the database; legacy Russian records display correctly after switching to English
- **GitHub Wiki** — repaired bilingual page links (`en/`, `ru/`)

### Changed

- **Header toggles** — language (flag) and Live/Static (radio/pause) are **single icons on all screen sizes**, not only on mobile

### Docs

- **Product Hunt** embed in `README.md` and `README.ru.md`
- Scripts to publish bilingual wiki and sync GitHub release notes

### Upgrade

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build dashboard
```

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/

## 🇷🇺 Русский

### Исправлено

- **Список уведомлений** — сообщения формируются из **заготовленных i18n-шаблонов по типу события**, а не из сырого текста в БД; старые записи на русском корректно отображаются после переключения на English
- **GitHub Wiki** — исправлены ссылки на двуязычные страницы (`en/`, `ru/`)

### Изменено

- **Переключатели в шапке** — язык (флаг) и Live/Static (радио/пауза) — **одна иконка на всех размерах экрана**, не только на мобильных

### Документация

- Виджет **Product Hunt** в `README.md` и `README.ru.md`
- Скрипты публикации двуязычной wiki и синхронизации release notes на GitHub

### Обновление

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build dashboard
```

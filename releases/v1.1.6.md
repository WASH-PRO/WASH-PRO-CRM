## 🇬🇧 English

### Added

- **Telegram bot types** — Management, Service, Information; command checkboxes auto-filled from preset
- **Information bot** (v1.0) — public menu: news, mode prices, post occupancy, promotions; feed from CRM
- **Information** section (`/info-messages`) — news/promotions with HTML, image URL, publish date, queue, wash binding
- **MQTT outbox** — outbound command log with `message_id`, `set/ack` delivery confirmation, retry without ack
- **Notifications** — alert bot settings in CRM; unread indicator on bell; `mqttCredit` / `mqttCollection` checkboxes

### Changed

- **Telegram bot v2.9** — menu from CRM checkboxes, RBAC in keyboard, currency from CRM reference, template rebuild on save/restart
- **Notifications** — all event types respect settings; web/Telegram channels; no duplicates from MQTT service account

### Fixed

- Viewer in management bot no longer sees create/full-access buttons with limited permissions
- Bot reports showed ₽ when CRM used another currency

### Upgrade

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build
docker compose run --rm init-seed
```

Or **Settings → Software updates** in Dashboard.

## 🇷🇺 Русский

### Добавлено

- **Типы Telegram-ботов** — Управление, Сервисный, Информационный; чекбоксы команд по пресету
- **Информационный бот** — новости, цены, занятость постов, акции; лента из CRM
- **Раздел «Информация»** (`/info-messages`) — новости и акции с HTML, изображением, датой публикации
- **MQTT outbox** — журнал исходящих команд, подтверждение `set/ack`, повторная отправка
- **Уведомления** — настройка бота оповещений; индикатор на колокольчике; чекбоксы `mqttCredit` / `mqttCollection`

### Изменено

- **Telegram-бот v2.9** — меню по чекбоксам CRM, RBAC, валюта из справочника
- **Уведомления** — все типы событий, каналы web/Telegram

### Исправлено

- Viewer в боте не видит кнопки создания при ограниченных правах
- Отчёты бота показывали ₽ при другой валюте в CRM

### Обновление

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build
docker compose run --rm init-seed
```

Или **Настройки → Обновления ПО** в Dashboard.

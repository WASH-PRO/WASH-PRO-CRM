v1.1.22 — In-app help, menu & breadcrumbs · Справка, меню и крошки

**English** · **Русский**

---

## 🇬🇧 English

### Added

- **Built-in help** — fullscreen modal from the header (book icon): TOC by menu group, search, screen wireframes, examples, link to GitHub Pages docs (EN/RU)

### Changed

- **Menu** — `/system` moved from **Main** to **System** group and renamed **Information** (server metrics & stack versions)
- **Automation** — info-messages section renamed **Publications** (was “Information” / “Info messages”)
- **Breadcrumbs** — trail follows sidebar structure (`Group → Section`), e.g. Analytics → Usage, System → Information; fixed intermediate links
- **i18n (RU)** — short label for **Usage** in collapsed sidebar

### Upgrade

```bash
git pull
docker compose up -d --build dashboard
```

Set `APP_VERSION=1.1.22` in `.env` if needed.

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/

## 🇷🇺 Русский

### Добавлено

- **Встроенная справка** — полноэкранное окно из шапки (иконка книги): оглавление по группам меню, поиск, схемы экранов, примеры, ссылка на GitHub Pages (EN/RU)

### Изменено

- **Меню** — `/system` перенесён из **Главное** в группу **Система** и переименован в **Информация** (метрики сервера и версии стека)
- **Автоматизация** — раздел info-messages переименован в **Публикации**
- **Хлебные крошки** — цепочка как в меню («группа → раздел»), исправлены ссылки
- **i18n (RU)** — короткая подпись «Использование» в свёрнутом меню

### Обновление

```bash
git pull
docker compose up -d --build dashboard
```

При необходимости: `APP_VERSION=1.1.22` в `.env`.

Документация: https://wash-pro.github.io/WASH-PRO-CRM/ru/

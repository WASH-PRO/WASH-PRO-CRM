## English

### Fixed

- **Post detail** — state history matches MQTT section: 100 rows per page, single «Load more» button above the table, server-side pagination via `usePolling`
- **Software updates** — update-bridge reconciles stale active job before starting a new update

### Upgrade

Auto-update from Dashboard (**Settings → Updates**) or:

```bash
git pull
docker compose build dashboard update-bridge
docker compose up -d dashboard update-bridge
```

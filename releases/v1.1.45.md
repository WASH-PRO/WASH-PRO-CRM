v1.1.45 — Help: module docs stay in modules · Справка: help модулей только в модулях

---

## English

### Changed

- **In-app help** — removed per-module sections from CRM Help sidebar (catalog-driven `module:*` entries and iframe)
- Module help remains on each module page (`/modules/:id` → built-in `ui/help.html` / `ui/help.ru.html`)
- Opening help from a module route now shows the general **Modules** CRM section

### Upgrade

```bash
git pull
docker compose build dashboard
docker compose up -d dashboard
```

---

## Русский

### Изменено

- **Встроенная справка CRM** — убраны отдельные разделы по каждому модулю из сайдбара справки (каталог `module:*` и iframe)
- Справка модуля остаётся на странице модуля (`/modules/:id` → встроенные `ui/help.html` / `ui/help.ru.html`)
- При открытии справки с маршрута модуля показывается общий раздел **«Модули»** CRM

### Обновление

```bash
git pull
docker compose build dashboard
docker compose up -d dashboard
```

## 🇬🇧 English

### Added

- **MCP server** (`services/crm-mcp`) — stdio MCP for AI agents (Cursor): CRM entities, composite tools, Dynamic API MCP proxy, Telegram bots and post commands via bridge

### Cursor setup

```bash
cd services/crm-mcp
npm install && npm run build
```

Example: `services/crm-mcp/cursor-mcp.example.json`

```json
{
  "mcpServers": {
    "wash-pro-crm": {
      "command": "node",
      "args": ["dist/index.js"],
      "cwd": "/path/to/WASH-PRO-CRM/services/crm-mcp",
      "env": {
        "CRM_API_URL": "http://localhost:3001",
        "CRM_DASHBOARD_URL": "http://localhost",
        "CRM_LOGIN": "admin",
        "CRM_PASSWORD": "Admin123!"
      }
    }
  }
}
```

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/mcp/

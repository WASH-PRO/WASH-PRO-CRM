## 🇬🇧 English

### Changed

- **Table pagination** — **Back/Next** replaced with **Load more**; tables show **200 rows** initially, then load 200 more per click
- Applies to all `DataTable` instances: post history, status, MQTT, logs, notifications, finance, usage, references
- Footer label: `N rows · shown M`

Overview notifications widget stays compact (10 rows + load more).

### Upgrade

```bash
git pull
docker compose up -d --build --no-deps dashboard
```

## 🇷🇺 Русский

### Изменено

- **Пагинация таблиц** — «Назад/Далее» заменены на **«Загрузить ещё»**; старт с **200 записей**, догрузка по 200
- Все таблицы на `DataTable`: история поста, состояние, MQTT, логи, уведомления, финансы, справочники
- Подвал: `N записей · показано M`

Виджет уведомлений на Обзоре — компактное превью (10 строк).

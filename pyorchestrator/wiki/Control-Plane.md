## Страницы

| Раздел | Путь | Описание |
|--------|------|----------|
| Обзор | `/` | KPI, графики runs/нагрузки/ресурсов, health |
| Скрипты | `/scripts` | Библиотека объектов, фильтры, массовые действия |
| Редактор | `/scripts/:id` | Monaco, multi-file, запуск/остановка, live-логи |
| Группы | `/groups` | Организация скриптов |
| Расписания | `/schedules` | Cron / interval / webhook |
| Вебхуки | `/webhooks` | Входящие HTTP-триггеры |
| Оповещения | `/notifications` | Алерты по runs |
| Бэкапы | `/backups` | Создание и восстановление |
| Информация о системе | `/system` | Health, RAM/диск, сервисы |
| MCP сервер | `/mcp` | Документация MCP и статус |
| Пользователи и группы | `/users` | RBAC (только Administrator) |
| Настройки | `/settings` | Профиль, тема, язык |

## Роли

| Роль | Возможности |
|------|-------------|
| **Administrator** | Полный доступ, управление пользователями |
| **Developer** | Скрипты, редактор, расписания |
| **Operator** | Запуск/остановка, просмотр |
| **Viewer** | Только чтение |

## Локализация

Интерфейс: **русский** и **английский** (раздел **Настройки** → язык).

## Обновление в реальном времени

Dashboard и списки поддерживают polling (режимы live/static). WebSocket используется для логов активного run в редакторе.

## Тема оформления

Светлая / тёмная / системная — переключатель в шапке боковой панели.

## English

### Changed

- **Layout** — mobile drawer always shows labels; fixed width up to 20rem; scroll lock, Escape, auto-close on navigation
- **Help** — collapsible section nav on small screens; content gets full height after selection
- **Safe area** — `viewport-fit=cover`, insets on header, sidebar, login, modals, and confirm sheet
- **System** — memory/disk stats stack on narrow screens (<380px)
- **Update banner** — actions wrap on small screens; **PageHeader** — compact action buttons on mobile

### Upgrade

```bash
git pull
docker compose build dashboard
docker compose up -d dashboard
```

## Русский

### Изменено

- **Layout** — мобильное меню всегда с подписями; фиксированная ширина до 20rem; блокировка прокрутки, Escape, закрытие при переходе
- **Справка** — сворачиваемая навигация на узких экранах; после выбора раздела — контент на весь экран
- **Safe area** — `viewport-fit=cover`, отступы в шапке, сайдбаре, входе, модалках и подтверждениях
- **Информация** — статистика памяти/диска в колонку на экранах <380px
- **Баннер обновлений** — кнопки переносятся; **PageHeader** — компактные кнопки на мобиле

### Обновление

```bash
git pull
docker compose build dashboard
docker compose up -d dashboard
```

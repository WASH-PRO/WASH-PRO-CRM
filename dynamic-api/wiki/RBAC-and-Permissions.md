## Permissions

`view`, `create`, `update`, `delete`, `manage_users`, `manage_api`, `view_logs`

## System groups

| Group | Access |
|-------|--------|
| Super Admin | Full |
| Admin | Full |
| Editor | API management |
| Manager | API + logs |
| User | View only |

## Endpoint access types

- `public` — no auth
- `authenticated` — any logged-in user
- `group` — specific groups only

## 🇬🇧 English

### Added

- **MCP server** section (`/mcp`) — Dynamic API / PyOrchestrator switch; tools table; ready HTTP config for Cursor; nginx proxy `/api/mcp` and `/api/pyorch-mcp/mcp`
- **`scripts/seed-info-messages.mjs`** — bulk create scheduled news/promotions
- **MCP documentation** — [docs/mcp.md](https://wash-pro.github.io/WASH-PRO-CRM/en/mcp/), updated Dashboard, wiki, README

### Changed

- **Dashboard menu** — **Automation** group: Information, Telegram, MCP, Backups; **Status** moved to **Main**; references in a separate group
- **Information** — **Scheduled** with past date shown as **Published** (filter, sort, CSV)
- **Information bot v1.9** — single message (photo + caption); correct post busy/online status

### Fixed

- **Gray screen on navigation** — JS chunk retry, `RouteErrorBoundary`, 60 s polling timeout

### Upgrade

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build dashboard pyorch-bridge
```

### Добавлено

- **Раздел «MCP сервер»** (`/mcp`) — переключатель Dynamic API / PyOrchestrator; таблица инструментов; готовый HTTP-конфиг для Cursor; прокси nginx `/api/mcp` и `/api/pyorch-mcp/mcp`

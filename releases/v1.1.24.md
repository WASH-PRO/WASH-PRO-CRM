v1.1.24 — Stale failed update jobs · Устаревшие ошибки обновления

**English** · **Русский**

---

## 🇬🇧 English

### Fixed

- **Stale failed job on update card** — when the running version is already at or past the failed job’s target (e.g. v1.1.23 while an old v1.1.20→v1.1.22 failure remains in history), the error is auto-hidden
- **Localhost stuck “Command exited with code 1”** after a later successful or manual upgrade

### Upgrade

```bash
git pull
docker compose up -d --build update-bridge dashboard
```

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/

## 🇷🇺 Русский

### Исправлено

- **Устаревший failed job на карточке** — если работающая версия уже ≥ цели failed job (например v1.1.23, а в истории ошибка v1.1.20→v1.1.22), ошибка скрывается автоматически
- **Зависшая «Команда завершилась с кодом 1»** на localhost после ручного или более позднего обновления

### Обновление

```bash
git pull
docker compose up -d --build update-bridge dashboard
```

Документация: https://wash-pro.github.io/WASH-PRO-CRM/ru/

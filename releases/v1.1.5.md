## 🇬🇧 English

### Changed

- **DataTable pagination** — unified UX for all CRM tables:
  - **Per page** dropdown: **20 / 40 / 60 / 80 / 100** (default **20**)
  - **Back / Next** — navigate loaded rows
  - **Load more (N rows)** — fetch next batch without reloading the whole table
- **Post state history** — starts at 20 rows, load more on demand
- **MQTT & Cards** — API load-more (100) above table + DataTable footer pagination

### Upgrade

```bash
git pull
docker compose up -d --build --no-deps dashboard
```

Docs: [Dashboard tables](https://wash-pro.github.io/WASH-PRO-CRM/en/dashboard/)

## 🇷🇺 Русский

### Изменено

- **Пагинация DataTable** — единый UX для всех таблиц CRM:
  - **На странице:** **20 / 40 / 60 / 80 / 100** (по умолчанию **20**)
  - **Назад / Далее** — навигация по загруженным строкам
  - **Загрузить ещё (N записей)** — догрузка без вывода тысяч строк сразу
- **История состояний поста** — старт с 20 записей
- **MQTT и Карты** — подгрузка с API (100) + пагинация в подвале

### Обновление

```bash
git pull
docker compose up -d --build --no-deps dashboard
```

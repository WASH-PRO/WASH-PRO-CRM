## 🇬🇧 English

### Added

- **Modules system** — `modules-bridge` service in Docker Compose; catalog in `modules/registry.json`
- **Dashboard → Automation → Modules** (`/modules`) — search, filters (installed/status/category), pagination, load more
- **Module lifecycle** — install, start/stop, update, uninstall from GitHub repos via PyOrchestrator
- **Module settings UI** — iframe with auto-height, CRM theme sync, breadcrumbs on `/modules/:id`
- **Module cards** — clickable title and icon; icon-only action buttons with tooltips
- **VK Publisher** — publishes CRM **Publications** to VK with photo upload ([wash-module-vk-publisher](https://github.com/WASH-PRO/wash-module-vk-publisher))
- **Shared module UI kit** — `wash-module-ui.css`, `wash-module-sdk.js` (Overview / Settings / Logs tabs)
- **API** — module icons (`GET /icon/:id`), run logs (`GET /installed/:id/logs`), enriched status

- Updated: architecture, dashboard, embedded services, README

### Upgrade

Requires **PyOrchestrator** for running modules:

```bash
git pull
# .env: PYORCHESTRATOR_ENABLED=true
docker compose build --pull=false dashboard modules-bridge
docker compose up -d --build dashboard modules-bridge
# or full stack:
./scripts/start.sh
```

On Mac without Docker Hub: use `--pull=false` if base images are cached.

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/modules/

## 🇷🇺 Русский

### Добавлено

- **Система модулей** — сервис `modules-bridge` в Docker Compose; каталог в `modules/registry.json`
- **Dashboard → Автоматизация → Модули** (`/modules`) — поиск, фильтры (установка/статус/категория), пагинация, «загрузить ещё»
- **Жизненный цикл** — установка, start/stop, обновление, удаление из GitHub через PyOrchestrator
- **UI настроек модуля** — iframe с авто-высотой, синхронизация темы CRM, хлебные крошки на `/modules/:id`
- **Карточки модулей** — кликабельный заголовок и иконка; кнопки действий только иконками с подсказками
- **VK публикатор** — публикация **Публикаций** CRM во ВКонтакте с загрузкой фото ([wash-module-vk-publisher](https://github.com/WASH-PRO/wash-module-vk-publisher))
- **Общий UI kit модулей** — `wash-module-ui.css`, `wash-module-sdk.js` (вкладки Обзор / Настройки / Логи)
- **API** — иконки модулей, логи запуска, расширенный status

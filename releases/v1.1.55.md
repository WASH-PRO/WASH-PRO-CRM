v1.1.55 — Fix dashboard Docker build on Linux (rolldown) · Сборка dashboard в Docker на Linux

---

## English

### Fixed

- **Dashboard image build** — builder base image switched from `node:20-alpine` to `node:20-bookworm-slim` so Vite 8 / rolldown installs the glibc native binding. Fixes server auto-update failure: `Cannot find module '@rolldown/binding-linux-x64-musl'`.

Includes all fixes from **v1.1.54** (MQTT credentials no longer wiped on post form save).

### Upgrade

```bash
git fetch origin && git checkout v1.1.55
# or Dashboard → Settings → Software updates
docker compose build dashboard message-processor
docker compose up -d dashboard message-processor
```

## Русский

### Исправлено

Включает все исправления **v1.1.54** (MQTT-учётки поста больше не затираются при сохранении формы).


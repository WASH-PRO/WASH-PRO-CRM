## 🇬🇧 English

### Fixed

- **Dashboard Docker build** — removed unused `BookOpen` import in `HelpModal.tsx` (TS6133 broke `npm run build` in v1.1.28)

### Upgrade

```bash
git pull
docker compose build --pull=false dashboard
docker compose up -d --no-deps dashboard
```

On Mac without Docker Hub: use `--pull=false` if `nginx:alpine` is already cached.

Docs: https://wash-pro.github.io/WASH-PRO-CRM/en/

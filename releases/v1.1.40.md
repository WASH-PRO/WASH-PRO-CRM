## English

### Fixed

- **Compose env when sourced** — `crm-update-compose-env.sh` resolves project root via `BASH_SOURCE` / `DEPLOY_ROOT` (macOS/Linux)
- **Integrity repair** — mosquitto, modules-bridge, init-seed use full `$COMPOSE_FILES` (override, Redis, PyOrchestrator)
- **PyOrchestrator auto-update** — build step respects `$COMPOSE_FILES`
- **macOS host scripts** — `crm-update-sync-host-env.sh` portable `sed -i`
- **Docs sync** — README badge and configuration.md track package version

### Upgrade

1. **Settings → Software updates** → update to **v1.1.40**
2. Rebuild **update-bridge** if upgrading manually: `docker compose build update-bridge && docker compose up -d update-bridge`

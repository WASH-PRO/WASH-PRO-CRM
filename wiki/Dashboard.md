# Dashboard

Полное описание: [docs/dashboard.md](https://wash-pro.github.io/WASH-PRO-CRM/dashboard/)

## Меню

| Группа | Разделы |
|--------|---------|
| Главное | Обзор |
| Объекты | Автомойки, Посты, Текущее состояние |
| Карты | Скидочные, Сервисные, VIP |
| Аналитика | Использование, Финансы, Архив |
| Система | Уведомления, **Пользователи**, **Группы и права**, Бэкапы, Telegram, Валюты, Типы скидок, Логи |

Admin-разделы — только **Administrator**.

## Resources (сайдбар)

- **Dynamic API** → `:8080` + статус
- **PyOrchestrator** → `:8090` + статус
- Документация, GitHub

## Live-обновление

| Интервал | Страницы |
|----------|----------|
| 3 с | Текущее состояние (+ график) |
| 5 с | Обзор |
| 10–15 с | Остальные |

# Мастер настройки

Полная документация: [docs/setup-wizard.md](https://wash-pro.github.io/WASH-PRO-CRM/setup-wizard/)

После первого входа — `/setup` до завершения первичной настройки.

## Шаги

Старт → Инфраструктура → Объект → Посты (MQTT-логины) → Валюта → MQTT (синхронизация) → Справочники → Готово.

## Роли

- **Administrator / Operator** — полный мастер, завершение
- **Viewer** — только просмотр + подтверждение ознакомления

## Повторный запуск

`/setup?restart=1` или **Система → Мастер настройки**.

## MQTT

«Синхронизировать MQTT» — обновляет passwd и ACL Mosquitto (изоляция постов по serial).

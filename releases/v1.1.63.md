## English

### Fixed

- **Informational bot silent in Telegram** — Docker bridge on some hosts cannot reach `api.telegram.org` (502 / timeout from `pyorch-bridge`). Added `telegram-egress` (`network_mode: host`) and route Bot API calls through it; DNS/IPv4 hardening kept as fallback.

### Upgrade

```bash
# Dashboard → Settings → Software updates → CRM
# Then stop/start the informational Telegram bot
```

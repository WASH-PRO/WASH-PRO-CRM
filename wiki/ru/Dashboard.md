# Dashboard

Полное описание: [docs/dashboard.md](https://wash-pro.github.io/WASH-PRO-CRM/ru/dashboard/)

## Меню (v1.1.14)

| Группа | Разделы |
|--------|---------|
| **Главное** | Обзор, **Состояние**, **Система** |
| **Объекты** | Автомойки, Посты |
| **Данные** | MQTT |
| **Карты** | Скидочные, Сервисные, VIP, Инкассация |
| **Аналитика** | Использование, Финансы, Архив |
| **Справочники** | Режимы работы, Валюты, Типы скидок *(Admin)* |
| **Автоматизация** | **Информация**, **Telegram**, **MCP сервер**, Бэкапы *(Admin)* |
| **Система** | Уведомления, Пользователи, Группы, Настройки, Логи |

Admin-разделы — только **Administrator**. Профиль — в шапке (`/profile`). Мастер настройки — `/setup`.

## Язык интерфейса (v1.1.13+)

- **English** (по умолчанию) и **Русский** — меню, статусы, сообщения, логи UI
- Переключатель в **шапке** (флаги 🇺🇸/🇷🇺; одна иконка) и в **Настройках**
- Выбор сохраняется в браузере (`wash_locale`); пользовательские данные не переводятся

## Уведомления (v1.1.14)

В списке **Уведомления** и на Обзоре показываются **локализованные шаблоны по типу события** (`mqtt_credit`, `user_login`, `wash_created` и др.), а не сырой текст из БД. Параметры (сумма, логин, имя объекта) извлекаются из старых записей. Метки severity и CSV-экспорт следуют выбранному языку.

## Статус поста онлайн/оффлайн

На страницах **Посты**, **Состояние** и **карточка поста**.

- **Онлайн** — телеметрия за последние 30 с (`lastMessageAt`)
- **Офлайн** — нет данных или устарели

## Раздел «Информация»

Новости и акции для информационного Telegram-бота. Статус **«По расписанию»** с прошедшей датой публикации отображается как **«Опубликовано»** (v1.1.12).

Массовое создание тестовых записей:

```bash
node scripts/seed-info-messages.mjs
# COUNT=50 START_DELAY_MIN=5 INTERVAL_MIN=10
```

## MCP сервер (`/mcp`)

- Dynamic API и PyOrchestrator (если включён)
- HTTP URL и конфиг для Cursor
- Таблица зарегистрированных инструментов

См. [MCP](ru-MCP).

## Настройки устройства поста

На странице поста (`/posts/:id#device-settings`) или через **⚙** в списке постов:

- **Цены режимов** — work-modes 0–9, MQTT `set/prices`
- **Команды** — перезагрузки, зачисление, сервисные режимы → `set/command`
- **Префикс MQTT** (`dt_pref`, обычно `washpro`)
- **Логин/пароль MQTT** — в форме создания/редактирования поста

## Resources (сайдбар)

- **PyOrchestrator** → `:8090` + статус

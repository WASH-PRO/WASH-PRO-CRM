## 🇬🇧 English

### Added

- **Dashboard localization (EN/RU)** — full UI translation; **English** default; switcher in header (🇺🇸/🇷🇺 flags) and **Settings**; choice stored in `localStorage` (`wash_locale`)
- **Multilingual GitHub Pages docs** — `docs/en/` and `docs/ru/`; root `/` redirects by saved language
- **Wiki EN/RU** — `wiki/en/` and `wiki/ru/`
- **README** — English `README.md` + `README.ru.md`

### Changed

- **Mobile header** — compact language and Live/Static icon toggles; full controls on `sm+`

### Fixed

- **Information** — green **Published** badge for **Scheduled** items with a past publish date

### Upgrade

```bash
git pull
docker compose -f docker-compose.yml -f docker-compose.pyorchestrator.yml up -d --build dashboard
```
